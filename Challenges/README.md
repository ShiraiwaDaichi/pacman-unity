# プログラミング初心者向け課題集

## 概要

このディレクトリには、プログラミング初心者がパックマンゲーム開発を通じて段階的にスキルアップできる課題が含まれています。各課題は、実際にコードを書いて学習することで、プログラミングの基礎からゲーム開発の応用まで身につけられるように設計されています。

## 課題の進め方

1. **段階的に進める**: 課題01から順番に取り組んでください
2. **実際にコードを書く**: 提供されたコード例をコピーせず、自分で入力してください
3. **エラーを恐れない**: エラーが出たら、原因を調べて修正する練習をしてください
4. **実験する**: 課題完了後、パラメータを変更して動作を確認してください
5. **理解してから進む**: 理解できない部分があれば、前の課題に戻って復習してください

## 学習のポイント

### プログラミング基礎概念
- 変数とデータ型
- メソッドと関数
- 条件分岐とループ
- クラスと継承
- 配列とコレクション

- Transform と座標系
- Rigidbody と物理演算

### ゲーム開発概念

- 状態管理
- イベント処理

- ユーザビリティ

- 各課題には**推定所要時間**が記載されています
- **前提知識**を確認してから取り組んでください
- **学習目標**を意識して進めてください
- 分からない時は**ヒント**を参考にしてください
